YTS Subtitles Lite

A lightweight Kodi subtitle addon for downloading English (Normal and HI) subtitles from YIFY Subtitles.

To install this addon:

  1. Download the ZIP from: service.subtitles.yts_subs_lite.zip
  2. In Kodi, go to Add-ons > Install from zip file
  3. Select the downloaded file

For Kodi repository XML, see: addons.xml
